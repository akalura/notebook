Select or create a page to get started.

    ⚙ Admin

    Export All

    Download a complete backup of all notebooks, pages, and images.

    Restore All

    Replace all current data with a previously exported full backup.

    ⚠ This will overwrite ALL existing notebooks, pages, and images.

    Storage Info

    Save to Server

    Save a backup copy directly to the server's backup folder for safekeeping.

    Server Backups

    Previously saved backups on the server. You can download, restore, or delete them.

    ⚠ Danger Zone

    Permanently delete all pages, tabs, and images. This action cannot be undone.

    ⚠ This will erase ALL data from localStorage and IndexedDB. Make sure you have a backup before proceeding.

    Export Notebook

    Export a single notebook with its pages and images.

    Import Notebook

    Import a backup as a new notebook without affecting existing data.

    Notebook Info

    Global Labels

    Labels defined here are available across all notebooks. They appear in the label dropdown with a ⊛ indicator.

    Label Macros

    Assign plugins to labels. When a page has a label with a macro, the plugin can be executed on that page's content.
